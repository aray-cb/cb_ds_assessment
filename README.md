# Data Science / Data Engineering Exercise

## Overview

This exercise consists of two short problems designed to assess your Python and pandas skills. The individual tasks will be given to you during the interview process.

* `exercise_1.py`: Python exercise
* `exercise_2.py`: pandas exercise

---

## Setup

Install dependencies:

```
pip install -r requirements.txt
```

---

## Data

The dataset for Exercise 2 is located at:

```
data/sat_scores.csv
```

Each row represents a student's SAT section score (Math or Reading & Writing) along with basic metadata.

---

## Instructions

* Complete the exercises given during the interview.
* You can run the files directly to test your solutions.
* Focus on clarity, correctness, and readability.

---

## Notes

* You are welcome to use your preferred IDE or development environment.
* If you prefer working in notebooks, feel free to convert the exercises to a `.ipynb` file.
* Please ensure your final solutions are complete and easy to follow.
